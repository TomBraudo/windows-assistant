=================================================
    WINDOWS AGENT - GUI VERSION QUICK START
=================================================

STEP 1: Install Dependencies
-----------------------------
Open terminal in this folder and run:

    pip install -r requirements.txt

This will install:
- customtkinter (GUI framework)
- litellm (LLM client)
- pycaw, comtypes (Windows system control)
- python-docx, python-pptx (document creation)
- python-dotenv (environment variables)
- requests (web APIs)


STEP 2: Configure API Keys
---------------------------
Option A - Use the GUI:
  1. Run: python main_gui.py
  2. Go to "Settings" tab
  3. Enter your API keys
  4. Click "Save Settings"

Option B - Edit .env file:
  Create/edit .env file with:
    GROQ_API_KEY=your_groq_key_here
    OPENROUTER_API_KEY=your_openrouter_key_here
    SERPAPI_API_KEY=your_serpapi_key_here
    SAFE_MODE=True


STEP 3: Launch the GUI
-----------------------
    python main_gui.py


ALTERNATIVE: CLI Mode
---------------------
To use the original command-line interface:
    
    python cli.py


=================================================
                TROUBLESHOOTING
=================================================

Problem: "ModuleNotFoundError: No module named 'customtkinter'"
Solution: Run: pip install customtkinter

Problem: "API Key not found"
Solution: Go to Settings tab and enter your API keys

Problem: GUI is too small/large
Solution: Edit main_window.py line 18:
    self.geometry("900x700")  # Change dimensions

Problem: Agent not responding
Solution: Check terminal for error messages, verify API keys


=================================================
                  USAGE TIPS
=================================================

1. Press Ctrl+Enter to send messages quickly
2. Enable Safe Mode in Settings for confirmation prompts
3. Check the terminal window for debug output
4. The CLI version (cli.py) is useful for testing tools
5. All conversation history stays in the GUI until you close it
